PortalHacienda Readme

PortalHacienda

Un paquete de interfase a la API del Portal de Datos del Ministerio de Hacienda

  • Buscar series en la descripcin de los meta-datos
  • Descargar las series directamente de la API del Portal
  • Extender y proyectar rpidamente series descargadas (funcionalidad muy bsica)

Instalacin

Para instalar el paquete en R ejecutar:

# Stable
# install.packages("PortalHacienda")

Versin de desarrollo:

# Development
# install.packages("devtools")
# devtools::install_github("fmgarciadiaz/PortalHacienda")

Nota: Debe instalarse previamente el paquete devtools para permitir la descarga desde github.

Bajar serie de tipo de cambio con Get y extender 12 perodos con Forecast (usa modelo auto-detectado del paquete forecast y extiende segn la frecuencia detectada, das, meses o aos). Luego hacer un plot sencillo.


TCN <- Forecast(Get("174.1_T_DE_CATES_0_0_32" , start_date = 2000), 12)       
#> Descagando series...
#> [1] "Cargada/s las series..."
#> [1] "Cargados 243 datos, desde 2000-01-01 hasta 2020-03-01 Periodicidad estimada: monthly"
#> Registered S3 method overwritten by 'quantmod':
#>   method            from
#>   as.zoo.data.frame zoo
#> [1] "Serie extendida 12 perodos, usando el modelo auto detectado: ARIMA(4,2,1)(0,0,2)[12]"
# Mostrar resultado
plot(TCN , main = "Tipo de Cambio Nominal ($/USD)")

plot(Get("6.2_AGCS_2004_T_39,6.2_IM_2004_T_23,6.2_C_2004_T_12") , legend.loc = "topleft" , main = "VAB sectorial ($ de 2004)")
#> Descagando series...

En caso de cargar varias series y desear proyecciones automticas, utilizar la variante vectorial de Forecast, que es vForecast:

TCN <- vForecast(Get("120.1_PCE_1993_0_24,120.1_ED1_1993_0_26"),12)
#> Descagando series...
#> [1] "Cargada/s las series..."
#> [1] "Cargados 64 datos, desde 1986-01-01 hasta 2017-01-01 Periodicidad estimada: yearly"
#> [1] "Serie extendida 12 perodos, usando modelo auto detectado"

Notas

Las series se cargan en formato XTS, con la periodicidad auto-detectada por el paquete xts. La periocidiad es usada por la funcin Forecast para correcta deteccin de estacionalidad y lags. Forecast agrega dems intervalos de confianza del 95%. Las series diarias tienen un tope de 1000 datos (dado el lmite actual de la API)
